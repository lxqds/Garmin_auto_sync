# 佳明直连接入指南（Garmin Connect 自动同步）

本指南对应已实现的 `garmin_sync.py`。它用 `garminconnect` 社区库**自动登录你的佳明账号**，
每天拉取：① 新运动活动（落 `garmin-data/inbox/` → 复用 `watch_and_analyze.py` 自动分析）；
② 每日健康摘要（睡眠/HRV/静息心率/身体电量/训练准备度/压力/步数 → `health_daily.md`）。

---

## 0. 合规与风险须知（先读）
- `garminconnect` 是对 Garmin Connect 的**社区逆向封装**，并非官方公开 API。用于拉取**自己账号**数据属个人用途，
  但接口可能随 Garmin 改版失效，请控制频率、勿高频猛刷。
- 令牌/密码仅存本地项目目录（已在 `.gitignore` 屏蔽），**切勿分享或提交到任何地方**。
- 开 MFA 时首次登录需在终端输入一次验证码；之后令牌（约 1 年有效、自动刷新）留存，无需再输。

---

## 1. 安装依赖（已为你装好，可跳过）
```bash
# 受管 venv（推荐）
C:/Users/user/.workbuddy/binaries/python/envs/default/Scripts/pip install garminconnect
```
> 全局环境也可：`pip install garminconnect`

---

## 2. 填写账号凭据
编辑 `garmin-data/.garmin_config.json`（首次运行 `auth` 会自动生成模板）：
```json
{
  "email": "你的佳明账号邮箱",
  "password": "你的佳明账号密码",
  "cookie_order_token": "",
  "cookie_jwt_fgp": ""
}
```
保存后，**不要**提交此文件（已 gitignore）。

### 不想存密码？用 cookie 免密方式
1. 浏览器登录 connect.garmin.com，打开开发者工具 → Network/Application → Cookies。
2. 复制 `orderToken` 和 `JWT_FGP` 两个 cookie 值。
3. 把上面 email/password 清空，填入 `cookie_order_token` / `cookie_jwt_fgp`。
> 这种方式连密码都不碰，但 cookie 过期后需重新复制。

---

## 3. 首次授权（auth）
```bash
python garmin_sync.py auth
```
- 若开 MFA，终端会提示 `请输入 MFA 验证码：`，从验证器输入一次即可。
- 成功后令牌保存到 `garmin-data/.garmin_tokens/garmin_tokens.json`（权限 0600）。
- 之后 `fetch` 只带令牌运行，**无需再输入密码/MFA**。

---

## 4. 每日拉取（fetch）
```bash
# 拉取近 30 天
python garmin_sync.py fetch

# 自定义回溯天数
python garmin_sync.py fetch --days 14
```
脚本会：
1. 用令牌登录（失效则自动用密码重登）。
2. 下载近 N 天、尚未处理过的活动原始文件（.fit/.tcx）落 `inbox/`。
3. 自动调用 `watch_and_analyze.py --once` 生成单次报告 + 趋势汇总 `summary.md`。
4. 拉取每日健康摘要，写入 `health_records.json` + 人类可读表 `health_daily.md`。

---

## 5. 查看状态
```bash
python garmin_sync.py status
```
显示：配置是否填写、令牌是否存在、上次拉取日期、已处理活动数/健康天数。

---

## 6. 无凭证自检
```bash
python garmin_sync.py test
```
不联网、不需要账号，验证"活动下载落盘 + 健康解析"逻辑是否正常（已验证通过）。

## 7. 真正零手动：挂每日定时任务
数据获取已是自动的（Garmin→本脚本）。想每天定时跑 `fetch`：
- **WorkBuddy 自动化**：让我帮你建一个每日定时任务，到点自动 `fetch`。
- **Windows 任务计划程序**：
  - 操作：`C:\Users\user\.workbuddy\binaries\python\envs\default\Scripts\python.exe`
  - 参数：`garmin_sync.py fetch`
  - 起始于：`D:\DATA\workbuddy\2026-07-22-11-29-35`
  - 触发器：每天 08:00（你到公司后，数据已同步完）。

---

## 8. 与 Strava 方案的关系
- **Strava 链路（`strava_sync.py`）**：仅运动活动，需 Garmin→Strava 官方同步，最合规。
- **本佳明直连**：运动 + **每日睡眠/HRV/身体电量/静息心率/训练准备度**，最契合你"规范作息"的目标。
- 两者可**并存**：Strava 管运动、佳明直连管健康；也可只用本方案（更全面）。

---

## 9. 隐私与清理
- 不再使用时，删除 `garmin-data/.garmin_tokens/` 与 `.garmin_config.json` 即可撤销本地授权。
- 手表数据所有权始终在你手中；本程序只做本地读取与归档，不上传任何第三方。
